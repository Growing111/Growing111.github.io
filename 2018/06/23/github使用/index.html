#正文

一、github里的watch,star,fork是什么？

watch翻译是观察，这里可以看作是观看，默认是没有观看的，
你可以选择观看中，然后只要这个这个项目发生变动，你个人中心就会有通知。
star翻译是星，这里解释为点赞，github里有个列表，里面会记录你所有点赞的项目。
这个就跟qq的收藏差不多的效果。
fork翻译为餐叉，在github里的作用是拷贝一份项目到你的github上。
